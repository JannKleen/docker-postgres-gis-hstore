## Postgres GIS Hstore

## Usage

### Postgres 9 (previous)

#### Build postgres 9 container
```bash
make postgres-9-build
```

#### Push it to the hub with tag 9
```bash
make postgres-9-push-tag-9
```

#### Push it to the hub with tag latest
```bash
make postgres-9-push-tag-latest
```
